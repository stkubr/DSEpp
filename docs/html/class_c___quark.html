Detailed Description

The Quark Dyson-Schwinger equations in general form.

It uses iterative Power Method to obtain the solutions of the equations the only input into equations is two-quark scattering kernel, provided by C_AbstractKernel * Kernel pointer through linkToKernel()

C_Quark_parameters params define the technicalities like: the number of integration points and etc.
